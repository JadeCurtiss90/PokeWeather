PokeWeather is a simple GUI application that polls the National Weather Service API and presents the forecast for any city/state pair in the United States. No installation required!

This is just a fun little side project, but if you have any recommendations or requests, let me know and I'll see what I can do!

Big thanks to Max Fleishman for the spritework and window frames! Check out more of his stuff over at http://www.maxfleishman.com/!
Also big thanks to the National Weather Service, Nominatim, and Geolocation DB for their powerful public APIs that made this possible!
